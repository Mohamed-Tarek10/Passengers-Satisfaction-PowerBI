# ✈️ Airline Passenger Satisfaction Analysis

A complete data analysis project exploring customer satisfaction among airline passengers using a dataset of **98,619 records**. The project investigates what factors influence satisfaction, how service ratings vary, and how delays, travel type, and flight class affect the passenger experience.

---

## 📘 Overview
This project analyzes passenger demographics, airport information, travel characteristics, and multiple service rating metrics to identify trends and insights related to airline satisfaction.  
Visualizations and dashboards were created using **Python**, **Power BI**, and **Tableau**.

---

## 📂 Dataset Description
**File:** Airline Passengers Satisfaction Dataset  
**Rows:** 98,619  
**Columns:** 37  

### Main Variables
- **Passenger Info:** ID, Gender, Age, Nationality  
- **Airport Info:** Airport Name, Country, Continent  
- **Flight Details:** Flight Distance, Class, Type of Travel, Delays  
- **Service Ratings:** Cleanliness, Food & Drink, Wifi, Seat Comfort, Baggage Handling  
- **Target:** Satisfaction (Satisfied / Neutral or Dissatisfied)
